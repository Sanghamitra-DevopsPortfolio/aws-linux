# Project 1: Launching AWS EC2 and Hosting Web Server
Step 1:Creating AWS account
-Signed up on Amazon Web Services.
-Enabled free tier.

---
Step 2:Launch EC2 Instance
-Chose Ubuntu server 22.04.
-Instance type: t3.micro.
-Created key pair.
-Allowed SSH and HTTP in Security group.

---
Step 3:Connect to EC2
-Used EC2 Instance Connect

---
Step 4:Linux Commands Practiced
-File system navigation
-User and permission management
-Process monitoring
-Disk usage
-Logs and troubleshooting

---
Step 5:Install Nginx
Commands used:
sudo apt update sudo apt install nginx

---
Step 6:Troubleshooting
Faced 504 Gateway Timeout.
Solved by enabling HTTP port 80 in security group.

---
Step 7:Outcome
Successfully deployed web server and accessed it via public IP.
